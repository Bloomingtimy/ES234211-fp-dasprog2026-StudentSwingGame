## Description

A simple Tic Tac Toe game developed using Java Swing GUI. The application supports user login, gameplay, player statistics, and Top 5 leaderboard.


## Features

- User login system
- MySQL database connection using JDBC
- Tic Tac Toe gameplay
- Win / Lose / Draw detection
- Score calculation
- Player statistics
- Top 5 scorers leaderboard


## Technology Used

- Java
- Java Swing
- MySQL
- JDBC


## Database

Database:
game_project

Table:
players


## How to Run

1. Create database using database/schema.sql
2. Open project in IntelliJ IDEA
3. Add MySQL Connector/J
4. Configure DatabaseManager.java
5. Run Main.java


## Project Structure

Main:
Starts application

DatabaseManager:
Handles database connection

Player:
Stores player data

PlayerService:
Handles database operations

GameLogic:
Handles Tic Tac Toe rules

LoginFrame:
Login interface

MainMenuFrame:
Main navigation

GameFrame:
Game interface

StatisticsFrame:
Player statistics

TopScorersFrame:
Leaderboard
